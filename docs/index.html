Critical Engineering — Heltec V3 — 868 MHz LoRa

Dead
Reckoning

A device that maps the invisible. It passively counts every smartphone that broadcasts its location history into the surrounding air — while silently relaying LoRa mesh traffic with no SIM, no carrier, no cloud.

Pre-release — firmware under active development

Web Serial unavailable. Use Chrome or Edge on desktop.

Connect your board via USB-C before clicking. Select the port labelled CP2102 or USB JTAG.

How to flash

01

Connect the board

Plug your Heltec V3 into your computer with a USB-C cable.

02

Click Install

The browser will ask you to select a serial port from a small dialogue.

03

Select the port

Choose the entry labelled CP2102 or USB JTAG/Serial and click Connect.

04

Wait ~60 seconds

The firmware flashes automatically. The OLED splash screen confirms success.

05

Deploy

Unplug and power from any USB-C source — powerbank, mains, or battery.

How it works

Core 0 — 868 MHz

LoRa mesh relay

Listens for LoRa packets from other nodes and relays them onward. No SIM, no internet, no origination — this node is a witness and a relay only.

Core 1 — 2.4 GHz

Probe sniffer

Passively captures 802.11 probe request frames — the involuntary broadcasts every smartphone emits when scanning for known networks. MACs are SHA-256 hashed. Raw addresses are never stored.

Display

OLED readout

Rotates every 5 seconds between mesh stats, probe counts, and a project statement. Hold the PRG button to freeze a screen. Hold 2 seconds to clear session data.
